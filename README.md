# Nail — 네일샵 예약/스케줄 관리 (Calendar-first)

네일샵 운영을 **캘린더 중심**으로 단순화하는 예약/스케줄 관리 서비스입니다.  
초기에는 내부 예약을 안정적으로 운영하고, 이후 **네이버 예약을 API로 동기화**해 캘린더에 정확히 반영합니다.  
장기적으로는 **웹 결제 → 자동 확정**까지 연결합니다.

## 핵심 기능
- **Day 캘린더 UI**(Google Calendar 스타일): 날짜 이동, 타임그리드, 예약 블록 표시
- **예약 상태 관리**: 가예약(PROVISIONAL) / 확정(CONFIRMED) / 취소 등
- **운영 자동화(예정)**: 가예약 3시간 미입금 시 자동 취소 + 알림
- **옵션 확장 구조**: 예) 젤 제거 +30분(현재), 추후 옵션 추가 가능
- **외부 연동(예정)**: 네이버 예약 webhook/polling 수신 → payload 원문 저장 → 내부 예약 매핑(멱등)

## 운영 정책(v1)
- Timezone: **Asia/Seoul (KST)**
- 예약금: **30,000원**
- 입금(또는 결제 성공) 확인 시 **즉시 확정**
- 가예약은 **3시간 내 미입금 시 자동 취소**
- 전일 22:00: 자동 취소가 아니라 **내일 확정 스케줄 가시화/락**을 위한 운영 기능

## 개발 단계
1) **Phase 1**: 캘린더 UI + 내부 예약 CRUD  
2) Phase 2: 만료/웨이팅/알림 자동화  
3) Phase 3: 네이버 예약 동기화(표준화 + payload 보존)  
4) Phase 4: 결제 도입

## 설계 원칙(중요)
- 하드코딩 금지(예약금/만료시간/22:00 규칙/옵션 시간 등은 설정/DB로 관리)
- 외부 연동은 **Adapter/Plugin** 구조로 확장 가능하게
- 외부 데이터는 **원문 payload(JSON) 보존**(필드가 늘어나도 재매핑/재처리 가능)
